For example, this is how we can define a function that computes and returns the empirical mean (average) of a numeric vector: ``` r my_mean_function = function(y) { n = length(y) sum = 0 for(i in 1:n) sum = sum + y[i] mean = sum / n return(mean) } ``` We can now use that function inside R for estimating means on any input vector `y` for which the the operators `+` and `/` are defined. ``` r # generate random vector x = rnorm(1000, mean = 23.42, sd = 4) # calculate mean with our function our_mean = my_mean_function(x) our_mean ``` ## [1] 23.43136 ``` r # compare with R's built in function mean(x) ``` ## [1] 23.43136
